페르시아어 타자 연습 & 단어장

⚠️ 필독 주의사항

1. 학습 기록은 사이트를 종료하거나 새로고침하면 초기화됩니다.

2. 오타를 방지하기 위해 반드시 윈도우/맥 설정에서 '페르시아어(표준)' 자판을 설치하고 연습하세요.

(※ 일반 페르시아어, 혹은 Mac의 'QWERTY' 자판을 쓰면 기호 배열이 맞지 않습니다! 꼭 '표준'으로 설치하세요.)

▶ Windows 페르시아어(표준) 키보드 설치 방법 보기
1. 윈도우 [설정](Win키+I) ➔ [시간 및 언어] ➔ [언어 및 지역] 접속
2. '기본 설정 언어'의 [언어 추가] 버튼 클릭
3. 검색창에 '페르시아어' 검색 후 설치
4. 키보드 옵션에서 반드시 '페르시아어(표준)' 레이아웃을 선택하세요.
5. 키보드 전환 단축키: Win키 + Space bar
▶ Mac 페르시아어(표준) 키보드 설치 방법 보기
1. 좌상단 [시스템 설정] ➔ [키보드] ➔ [텍스트 입력(입력 소스)] 편집 클릭
2. 좌하단 [+] 버튼 클릭
3. 검색창에 '페르시아어' 검색 후 반드시 '페르시아어(표준)' 추가
4. 키보드 전환 단축키: Ctrl + Space bar
